# Ground Up Monitoring System

This repository contains the monitoring system for the Ground Up Factory, structured into cleanly separated directories for ease of development.

---

## Codebase Map & Directory Structure

```text
ground-up-monitoring/
│
├── backend/                         # BACKEND API (Python / FastAPI)
│   ├── middleware/                  # Security/cors and JWT validation
│   ├── models/                      # SQLAlchemy Database models (Room, Sensor, Reading, Alert)
│   ├── routes/                      # API endpoints (rooms, sensors, alerts)
│   ├── services/                    # Core logic services (e.g., eWeLink API integration)
│   ├── config.py                    # Configurations (DB connection, JWT secrets)
│   ├── database.py                  # Database engine initialization
│   ├── main.py                      # FastAPI App entry point & IoT simulator thread
│   ├── worker.py                    # Telemetry processor & alert generator worker
│   └── requirements.txt             # Python dependencies
│
├── web/                             # WEB DASHBOARD (HTML / JavaScript)
│   └── index.html                   # HTML interactive web dashboard
│
├── mobile-shared/                   # SHARED MOBILE APP (React Native / Expo)
│   ├── assets/                      # Application icons and images
│   ├── src/                         # JavaScript source code
│   │   ├── navigation/              # App screens navigation
│   │   ├── screens/                 # Mobile pages (Login, Dashboard, Sensors, Analytics)
│   │   └── services/                # Axios API request clients
│   ├── App.js                       # Root React Native React component
│   ├── app.json                     # Expo manifest configurations
│   ├── eas.json                     # Expo cloud build configurations
│   └── package.json                 # Node dependencies
│
├── mobile-android/                  # NATIVE ANDROID BUILD (Java/Kotlin Gradle Project)
│   └── (Generated by running `npx expo prebuild` from the mobile-shared directory)
│
├── mobile-ios/                      # NATIVE IOS BUILD (Objective-C/Swift Xcode Project)
│   └── (Can be generated by running `npx expo prebuild --platform ios` from macOS/Linux)
│
├── .env                             # Local environment secrets
├── .python-version                  # Python version lock file (3.12)
└── Procfile                         # Heroku/DigitalOcean deployment commands
```

---

## Technologies & Languages Used

### 1. Backend (`backend/`)
* **Language**: Python (v3.12)
* **Libraries**: FastAPI (API layer), SQLAlchemy (ORM), Uvicorn (ASGI server)
* **Database**: PostgreSQL (Cloud)

### 2. Web Dashboard (`web/`)
* **Language**: HTML5 / JavaScript (Vanilla)
* **Description**: Renders real-time telemetry tables and status warnings from the backend API.

### 3. Mobile Shared App (`mobile-shared/`)
* **Language**: JavaScript / React Native
* **Framework**: Expo SDK (cross-platform deployment)
* **Libraries**: Axios (REST requests), React Navigation
